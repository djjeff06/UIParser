README:
*If not first time running, skip to step 4
1. install https://chrome.google.com/webstore/detail/allow-control-allow-origi/nlfbmbojpeacfghkpbjhddihlkkiljbi?hl=en-US

2. Download and install npm to the system https://nodejs.org/en/download/

3. in cmd: enter npm install http-server -g

4. cd to file directory of sample.xml

5. in cmd: enter http-server

6. change string in line 18 of htmlparser.html to any ip address shown in cmd after step 4 then append "/sample.xml".

7. If step 6 did not work, change line 18 of htmlparser.html to "sample.xml".

8. ctrl-c to exit server
